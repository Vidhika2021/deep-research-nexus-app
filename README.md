## Local Setup

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Set your API key:
    ```bash
    set AGENTS_API_KEY=your_key_here
    ```
4.  Run the app:
    ```bash
    uvicorn app:app --reload
    ```
